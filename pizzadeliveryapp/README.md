
# Pizza Delivery App

This is a simple pizza delivery app for Android.

## Project Structure

- `app/src/main/java/com/example/pizzadelivery` - Contains the Java source code for the app.
- `app/src/main/res` - Contains the resources for the app, such as layouts, strings, and images.
- `app/build.gradle` - The build script for the app.

## How to Run

1. Open the project in Android Studio.
2. Replace `"YOUR_API_KEY"` in `app/src/main/AndroidManifest.xml` with your Google Maps API key.
3. Run the app on an emulator or a physical device.
